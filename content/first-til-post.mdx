This is my TIL (Today I Learned) blog. It's a place where I write short posts
about things I learn day to day — programming concepts, tools, tricks, and
anything else worth remembering.

## Why a TIL blog?

Writing things down helps me remember them. Publishing them keeps me honest
about actually understanding what I'm writing. And maybe someone else finds
them useful too.

## The stack

This site is built with:

- **Next.js 15** with the App Router
- **TypeScript** for type safety
- **Tailwind CSS v4** for styling
- **MDX** for content — write in Markdown, embed React components when needed

Posts are just `.mdx` files in a `content/` directory. No CMS, no database.
